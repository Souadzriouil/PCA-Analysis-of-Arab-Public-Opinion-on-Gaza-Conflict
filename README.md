# 📌 Overview

This project explores **Arab public opinion on the Gaza conflict** using **Principal Component Analysis (PCA)** and statistical analysis techniques.

The objective is to identify the main factors influencing opinions across multiple Arab countries while reducing data dimensionality and preserving the most significant information.

The analysis combines:

- Exploratory Data Analysis (EDA)
- Correlation analysis
- PCA dimensionality reduction
- Statistical interpretation
- Data visualization

to better understand the structure of public opinion data.

---

# 🎯 Project Objectives

- Perform Exploratory Data Analysis (EDA)
- Analyze relationships between variables
- Apply PCA for dimensionality reduction
- Identify influential political and social factors
- Visualize multidimensional data
- Extract meaningful statistical insights

### Source
Arab Center for Research & Policy Studies

### Dataset Information

| Attribute | Details |
|---|---|
| Sample Size | ~8000 respondents |
| Countries | 16 Arab countries |
| Analysis Type | Multivariate Statistical Analysis |

---

# 🧾 Variables

| Variable | Description |
|---|---|
| SMPEU | Support of US military/political actions |
| MDAGAEI | Lack of Arab government action |
| AN | Normalization agreements |
| SGOI | Western government support |
| MDADONUI | UN inaction |
| MDPPLAPI | Palestinian authority position |
| Autre / NS | Other responses |

# 📈 Key Results

| Result | Insight |
|---|---|
| PCA Variance | First 2 principal components explain ~87.9% of total variance |
| Correlation | Strong relationship between AN and MDADONUI |
| Distribution | SMPEU variable follows a near-normal distribution |
| Dataset Balance | Observations are balanced across countries |

---

# 🧠 Statistical Insights

- Political and international factors strongly influence public opinion
- PCA successfully reduced data complexity while preserving key information
- Two principal dimensions are sufficient to represent most variability
- Correlation structures reveal strong relationships between geopolitical perceptions
